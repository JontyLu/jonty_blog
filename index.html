JavaScript 数组的方法

  2. 迭代器函数

    • foreach

      forEach方法接收一个函数作为参数，对数组中每个元素使用这个函数，只调用这个函数，数组本身没有任何变化

    • every

      every方法接受一个返回值为布尔类型的函数，对数组中的每个元素使用这个函数，如果对于所有的元素，该函数均返回true，则该方法返回true，否则返回false

    • some

      some方法也是接受一个返回值为布尔类型的函数，只要有一个元素使得该函数返回true，该方法就返回true
