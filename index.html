三、减少页面加载时间的方法

  1. 减少请求次数

    • CSS spirit（图片小精灵）
    • 合并js和css文档

  2. 压缩内容

  3. CSS和js位置（视觉上加快页面加载速度）

    • CSS放置在head里
    • js放置在文档末

  4. 使用CDN（内容分发网络）

    • 可以将静态内容放在这些CDN上，请求这些文件时会比较快

  5. DNS预解析

    • 拥有该属性的域名不需要用户地啊你连接就在后台解析了

总结：与DNS有关的前端优化中有两个部分

  1. 减少DNS的请求次数
  2. 进行DNS预解析

四、WebStorage和Cookies的区别

  1. WebStorage：容量存储更大
    • session storage：浏览器关闭后数据就丢失失效了
    • local storage：浏览器关闭了数据仍可以保存下来，永久存储永不失效
  2. cookies：大小首先，请求新页面时cookies会被发送过去，浪费带宽
