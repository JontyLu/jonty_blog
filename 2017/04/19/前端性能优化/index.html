这次的问题就是怎么加快页面访问的速度呢？

之后会慢慢地补充~

减少请求次数

  1. CSS Sprites(图片精灵)

    将多张图片组合成一个图片，请求次数就从[多]变成[1]了。

    前端用background-position:x y;来切割图片就好了

  2. 合并脚本，合并样式表

  • 脚本

使用CDN（内容分发网络）

  • 什么是CDN

    一组分布在多个不同地理位置的Web服务器，用户可以就近去的所需内容，可看做一种缓存代理

  • 能干嘛

    用户可以就近获得静态资源，假如你需要要jQuery框架，使用大公司提供的CDN服务会更快一点
