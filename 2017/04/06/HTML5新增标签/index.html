HTML5新增标签

发表于 2017-04-06 | 分类于 HTML | | 浏览 次

article

论坛的帖子、杂志或者新闻文章、博客或者其他独立的内容项目。

aside

表示页面的一部分，跟该页面其他内容关联性不强。一般用于显示一些目录、索引、或者广告宣传、作者介绍、简介内容等。

audio

  • 概念

    定义声音。如音乐或者其他的音频流。

  • 浏览器支持

    IE9+、FireFox、Chrome、Safari、Opera

  • 常用属性

    属性属性值注释
    srcurl音乐的url地址
    preloadpreload预加载（使用autoplay则该属性失效）
    looploop循环播放
    controlscontrols显示控制条
    autoplayautoplay自动播放

  • HTML DOM Audio 对象

    1. 对象属性
      • duration：获取总时长，单位是s
      • paused：被暂停返回true
      • ended：播放完毕返回true
      • muted：静音与否
      • volume：音量：0-1
      • currentTime：获取当前播放时间
      • startTime：返回起始播放时间
    2. 对象方法
      • load()：加载音频、视频软件
      • play()：加载并播放音频、视频
      • pause()：暂停
      • canPlayType(obj)：测试是否支持给定的文件
    3. 事件
      • loadstart：客户端开始请求数据
      • process：缓冲中
      • play：播放时
      • pause：暂停触发时
      • ended：播放结束
      • timeupdate：当前时间发生改变
      • canplaythrough：歌曲载入完成
      • canplay：缓冲至可播放

canvas

  • 概念：定义图形，如图表和图像。该标签只是图形容器，必须使用脚本来绘制图形。

  • 属性：只有height和width

  • 方法：getContext():返回的是绘制画布的环境

  • 绘制矩形、线条、弧形

command

没有浏览器支持

datalist

  • 概念：定义选项列表（下拉框）
  • 和select的区别
    datalist除了select的下拉选择之后，还可以往输入框输入值
  • 用法：需要和input标签绑定

embed

  • 定义嵌入的内容

  • 用途：可以用来插入各种多媒体：swf、Midi、Wav、AIFF、AU

  • 属性

    • src：文件路径
    • 还有controls、loop、height、width、volume（0-100）等

  • 跟object标签的区别

    五大浏览器都支持（国产可能embed不支持），Firefox是无视object，生效embed

footer

  • 概念：文档的页脚，跟article、aside类似

header

  • 概念：定义section和page的页眉，跟footer、article、aside类似

nav

  • 概念：定义导航链接的部分

  • 浏览器支持

    五大浏览器都支持（IE8以下不算入内）

  • 用法：如果有前后的按钮，就放在nav元素

output

  • 定义不同类型 的输出

  • 浏览器支持

    只有IE不支持

  • 参考此处

time

  • 概念：定义时间和日期
  • 浏览器支持：都不支持

video

  • 概念：定义视频，视频流

  • 浏览器支持

    吃了IE8以下，五大浏览器都支持

  • 方法和属性跟audio类似，可相互参考

还有其他新增的标签，可参考W3C此处
