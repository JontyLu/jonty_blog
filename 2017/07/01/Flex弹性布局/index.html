Flex弹性布局

发表于 2017-07-01 | 分类于 HTML

  • 什么是弹性布局

    它是一种伸缩布局盒模型，常用语设计比较复杂的页面，可以轻松实现屏幕和浏览器穿裤大小发生变化时保持元素的相对位置和大小不变。

  • 浏览器支持

    五大主流浏览器都支持

基本语法

行内元素也可以使用Flex布局

模型介绍

容器默认有两根轴，水平的主轴（main axis）和垂直的交叉轴（cross axis）

项目默认沿着主轴排列

属性

flex-direction属性

决定了主轴的方向，也就是项目的排列方向

flex-wrap属性

如果一条轴线放不下元素item，元素该如何换行

flex-flow属性

该属性是flex-direction和flex-wrap属性的简写形式

justify-content 属性

该属性定义了项目元素在主轴（可以理解为水平方向）上的对齐方式

align-item属性

该属性定义了项目元素在交叉轴（可以理解为垂直方向）上的如何对齐

align-content属性

多根交叉轴线的对齐方式

项目元素的属性

order属性

flex-grow属性

定义项目放大比例，默认为0，即空间充足，该项目也不放大

flex-shrink属性

定义了项目的缩小比例，默认为1，即空间不足，该项目将缩小

flex-basis属性

定义了在分配多余空间之前，项目占据的主轴空间

默认为auto，为项目的本来大小

设为跟width和height属性一样的值，则占据固定空间

align-self属性

允许单个项目与其他项目元素有不一样的对齐方式

auto为继承父元素的align-iitem属性，没有父元素则等同于stretch。
